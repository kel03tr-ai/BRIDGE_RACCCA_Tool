# BRIDGE & RACCCA Tool

An interactive tool for educators to evaluate AI tools and assess AI-generated output using the BRIDGE and RACCCA frameworks.

**Live Tool:** https://kel03tr-ai.github.io/BRIDGE_RACCCA_Tool/BRIDGE_RACCCA_Tool.html

## What is BRIDGE & RACCCA?
- **BRIDGE** is a checklist for selecting the right AI tool before generating content.
- **RACCCA** is a rubric for evaluating the quality of AI-generated output after generation.

## How to Use
Open the live link above in any browser — no installation needed. Designed for use during the Tool Tasting activity in professional development sessions.
